# Changelog

## Current

- Flask CRM app with login-protected lead, client, property, broker, meeting, follow-up, archive, backup, analytics, and assistant workflows.
- Sensitive-field encryption support with CLI migration tooling.
- Soft archive and restore flows for safer data handling.
- Property edit support from property cards and detail modal.
- Repository ignores runtime data, backups, caches, database files, and local secrets.
